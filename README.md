# Weather App

A simple weather application to check real-time weather information for any city.

**Live Demo:** [weather-today-by-dpnshuu.vercel.app](https://weather-today-by-dpnshuu.vercel.app/)

## What it does

- Search weather by city name
- Shows temperature, humidity, and wind speed
- Works on mobile and desktop
- Clean glassmorphism design

## Built with

- HTML, CSS, JavaScript
- OpenWeather API
